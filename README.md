### 🔓 Código Abierto y Transparencia
**HaxCore** es un proyecto de código abierto. Todos los scripts de automatización, la conexión del Bridge con Discord (Node.js) y la extensión del navegador están disponibles en las carpetas del paquete para que cualquiera pueda revisarlos, auditarlos o sugerir mejoras. No hay ejecutables ocultos ni procesos maliciosos.

---

### ⚙️ Instrucciones de Instalación
Al ser un cliente **Portable**, no requiere instalación en el sistema:

1.  Andá a la sección de **Releases** a la derecha de este repositorio.
2.  Descargá el archivo comprimido `HaxCore.zip`.
3.  **Extraé el contenido:** Es importante que lo descomprimas en una carpeta (por ejemplo, en el Escritorio) antes de abrirlo. No lo ejecutes directamente desde el ZIP.
4.  Ejecutá el archivo **`HaxCore.exe`**.
5.  Si Windows te pide permisos para el Bridge (Node.js), aceptalos para que el Discord RPC funcione correctamente.

---

### 🛠️ Estructura Técnica
* **Motor:** Basado en Iron Browser (Chromium) optimizado para baja latencia.
* **Bridge:** Comunicación mediante `discord-rpc` sobre Node.js.
* **Flags:** Lanzamiento con aceleración por hardware forzada y limpieza de caché automática.
